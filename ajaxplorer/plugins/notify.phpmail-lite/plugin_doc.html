Not active by default, this plugin adds notification features to notify some administrator by email of any action. Activate it via the GUI, and manually list the actions that you want to be notified of. This will dynamically add postprocessors to these action in the XML registry.

If you activate the "Enrich Shares" function, the "Share" panel will also provide the user with the ability to receive notification on upload and download performed on the repositories they share, as well as when a shared file is downloaded.
